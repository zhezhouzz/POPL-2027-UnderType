\section{Overview of Supplemental Materials}

The supplemental material is organized as follows.
\autoref{sec:tech:semantics} gives the complete operational semantics
of \langname{}.  \autoref{sec:tech:basic-typing} provides the full set
of \langname{}'s basic typing rules, qualifier typing rules, and
well-formedness judgments for context types and type contexts.
\autoref{sec:formalization-details} provides the details definition of
environment and capability.  A Rocq formalization of our
nondeterministic extensions to the core language, \langNDname{}, and
its type system, together with machine-checked proofs of the
fundamental theorem and soundness theorem, is available at
\url{https://anonymous.4open.science/r/POPL27-ContextType-SM-52EC}.
